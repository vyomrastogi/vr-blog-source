### IOC Container 

IOC container find the beans, wire the bean and maintains the lifecycle of a bean. 
There are two types 
- Beanfactory
  - Most basic container provided by spring
  - 
- ApplicationContext
  - advanced version of IOC container
  - I8n capabilities 
  - Provided webapplication context support
  - support for AOP
